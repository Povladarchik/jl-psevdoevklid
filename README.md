# Обобщение леммы Джонсона–Линденштраусса на псевдоевклидовы пространства

[![БММ](https://img.shields.io/badge/БММ-проект_247-1f4e79)](https://bmm.mca.nsu.ru/project/247)
[![XeLaTeX](https://img.shields.io/badge/LaTeX-XeLaTeX-008080)](#сборка-pdf)

Исследование псевдоевклидова аналога классической леммы Джонсона–Линденштраусса: насколько можно понизить сигнатуру пространства **R<sup>p,q</sup>** при сохранении квадратов расстояний с относительной ошибкой **ε**.

> Исходная постановка проекта: [Большая математическая мастерская, № 247](https://bmm.mca.nsu.ru/project/247).

---

## Мотивация

В евклидовом случае лемма JL утверждает, что **N** точек можно вложить в размерность

```text
O( log(N) / ε² )
```

с мультипликативной ошибкой **1 ± ε** на попарных расстояниях.

В псевдоевклидовом пространстве **R<sup>p,q</sup>** квадратичная форма знакопеременна:

```text
‖x‖² = x₁² + … + x_p² − x_{p+1}² − … − x_{p+q}²
```

Квадрат расстояния может быть положительным, нулевым или отрицательным; множество пар с **‖x − y‖² = 0** образует **световой конус**. Естественный вопрос: сохраняется ли возможность сильного (JL-подобного) понижения размерности?

---

## Основной результат

Для любого фиксированного **0 < ε < 1** любая **ε-изометрия**

```text
f : R^{p,q} → R^{p',q'}
```

(в смысле относительной ошибки по модулю квадрата псевдорасстояния) удовлетворяет

```text
p' + q'  ≥  C · min(p, q)
```

для некоторой абсолютной константы **C > 0**.

Иными словами, **сильное понижение сигнатуры до o(min(p, q)) невозможно** — в резком контрасте с евклидовой JL-оценкой.

### Определение ε-изометрии

```text
| ‖f(x) − f(y)‖² − ‖x − y‖² |  ≤  ε · |‖x − y‖²|
```

Такое определение корректно при любом знаке **‖x − y‖²** и влечёт сохранение светового конуса:

```text
‖f(x) − f(y)‖² = 0  ⇔  ‖x − y‖² = 0
```

---

## Идея доказательства

1. **Реализуемость «матриц светового конуса».**  
   Любая симметричная {0,1}-матрица **A** размера **n × n** (нули на диагонали) реализуема точками в **R<sup>n,n</sup>**: **A<sub>ij</sub> = 0** тогда и только тогда, когда **‖x<sub>i</sub> − x<sub>j</sub>‖² = 0**. Конструкция — через положительно определённую матрицу Грама **M = I + ε₀ B**.

2. **Нижняя оценка по счёту.**  
   Число матриц, реализуемых в пространстве сигнатуры **s = p + q**, ограничено сверху числом знаковых условий квадратичных полиномов (теорема Милнора–Тома / оценка Уоррена). При **s < C n** не все матрицы реализуемы.

3. **Перенос на JL.**  
   ε-изометрия сохраняет нулевые расстояния. Образы «плохой» конфигурации из **min(p, q)** точек дали бы реализацию в целевом пространстве малой сигнатуры — противоречие.
